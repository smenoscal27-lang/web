¿Cómo funciona el Clima Adaptativo?

La plataforma interactúa directamente con las APIs públicas globales de Open-Meteo y OpenStreetMap Nominatim para obtener el clima y nombre de tu ubicación actual mediante el navegador, ¡sin requerir ninguna clave de API! Al procesar el estado climático, inyecta clases en el elemento raíz para adaptar las variables CSS globales y rediseñar los colores del sitio.

El tema adaptativo cambia las variables de gradientes y colores de énfasis (`--accent-color`) para reflejar los estados naturales.
